## 分支管理

| 命令                    | 作用             |
| ----------------------- | ---------------- |
| git branch              | 查询本地分支     |
| git branch -r           | 查询远程仓库分支 |
| git branch [bname]      | 新建分支         |
| git branch -d [oldname] | 删除分支         |
|                         |                  |

## 分支管理

`git branch -vv` 查看分支追踪关系

`git branch`本地分支

`git branch -r`远程分支

`git branch [分支名]`

`git checkout (-b) [分支名]`切换分支

`git merge [被合并的分支]`合并分支
