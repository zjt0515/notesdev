## 朴素模式匹配算法

主串长度n
模式串长度m
字符串模式匹配：在主串中找到与模式串相同的子串，并返回其所在位置

找出主串中所有和模式串长度相同的子串 依次和 模式串匹配，共有n-m+1次

```c++
int index(string s, string t){
    int i = 1, n = length(s), m = length(t);
    while(i<n-m+1)
}
```
