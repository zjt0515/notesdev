## compilerOptions

编译器选项

| 属性             | 作用             |                  |
| ---------------- | ---------------- | ---------------- |
| target           | 选择编译版本     | es5、es6         |
| module           | 模块规范         | ESNext、CommonJS |
| rootDir          | 要编译的文件目录 |                  |
| outDir           | 编译后的存放目录 |                  |
| moduleResolution |                  |                  |
