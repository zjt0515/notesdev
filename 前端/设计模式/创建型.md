# 创建型

## 单例模式singleton pattern

> 确保对象只有一个
> 应用场景：全局性功能，保证对象只有一个，例如路由对象，store存储对象

1. 在初始化类时创建对象
2. 构造函数设置为私有，不允许外部调用
3. 提供获取实例方法getInstance

```java
public class Singleton {

    private static final Singleton INSTANCE = new Singleton();

    private Singleton() {}

    public static Singleton getInstance() {
        return INSTANCE;
    }
}
```
