## 引入css

| 引入方式 | 描述                                                     | 优缺                     |
| -------- | -------------------------------------------------------- | ------------------------ |
| 内联     | 在标签的style属性中                                      | 只对当前标签生效         |
| 嵌入     | 在页面head标签(html头部)下的style标签中                  | 只对当前html生效         |
| 链接     | <link rel="stylesheet" type="text/css" href="style.css"> | css代码就要放在css文件中 |
| 导入     | <style>     @import url(style.css); </style>             |                          |
