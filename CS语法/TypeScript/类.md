## Enum类

```ts
enum Color {
  Red, 	// 0
  Green	// 1
}
// 调用
Color.Green;
Color['Green'];
```

> [!note]
>
> 官方建议谨慎使用 Enum 结构，因为它不仅仅是类型，还会为编译后的代码加入一个对象。
> 由于 Enum 结构编译后是一个对象，所以不能有与它同名的变量
>
> Enum是类型：可以用枚举类型标注类型
> Enum结构也是一个值：编译后是一个对象
>
> ```ts
> // 编译后
> let Color = {
>   Red: 0,
>   Green: 1,
>   Blue: 2
> };
> // 使用as const实现的枚举
> const Bar = {
>   A: 0,
>   B: 1,
>   C: 2,
> } as const;
> ```
